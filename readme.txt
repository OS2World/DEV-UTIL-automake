Installation
------------

  Change drive to the drive pointed by %UNIXROOT%.

  Unzip automake-*-src.zip.

  CD into the directory.

  Then, do these.

      sh ./configure --prefix=/usr

      make

      make install

  Now, you can see that autoconf was installed into %UNIXROOT%/usr.
